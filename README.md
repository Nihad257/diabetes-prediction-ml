# Diabetes Prediction using Machine Learning

This project builds a Machine Learning model to predict whether a person is likely to have diabetes based on medical attributes. It demonstrates a beginner-friendly end-to-end ML workflow using Python.

---

## 📌 Objective
To develop a classification model that predicts diabetes outcome using patient health data.

---

## 📂 Dataset
**Pima Indians Diabetes Dataset** (UCI Machine Learning Repository)

---

## 🧪 Workflow Followed
1. Data loading using Pandas  
2. Data cleaning and handling missing values  
3. Exploratory Data Analysis (EDA) and visualizations  
4. Feature preparation for modeling  
5. Logistic Regression model building  
6. Model evaluation using Accuracy, Confusion Matrix, and Classification Report

---

## 📊 Model Performance
- **Accuracy:** 75%  
- Evaluated using precision, recall, and F1-score

---
