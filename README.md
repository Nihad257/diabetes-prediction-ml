Diabetes Prediction using Machine Learning

This project builds a Machine Learning model to predict whether a person is likely to have diabetes based on medical attributes. It demonstrates a complete beginner-friendly end-to-end ML workflow using Python.

📌 Objective

To develop a classification model that predicts diabetes outcome using patient health data.

📂 Dataset

Pima Indians Diabetes Dataset
(Referenced from UCI Machine Learning Repository)

🧪 Workflow Followed
Data loading using Pandas
Data cleaning and handling missing values
Exploratory Data Analysis (EDA) and visualizations
Feature selection and preparation for modeling
Logistic Regression model building


Model evaluation using:
Accuracy
Confusion Matrix
Classification Report

📊 Model Performance
Accuracy: 75%
Balanced evaluation using precision, recall, and F1-score
